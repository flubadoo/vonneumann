# vonneumann

Simple Von Neumann Machine simulator made using Scheme/Racket.
Input should be in the format of:

'''scheme
(define prog2
  '((start: input val)
    (load val)
    (skipzero)
    (jump continue)
    (halt)
    (continue: storei pointer)
    (load pointer)
    (add one)
    (store pointer)
    (jump start)
    (one: data 1)
    (val: data 0)
    (pointer: data table)
    (table: data 0)))
'''
